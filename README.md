### Vue JS Client (`vue-client`)

#### Project setup
```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn serve
```

#### Compiles and minifies for production
```
yarn build
```

#### Lints and fixes files
```
yarn lint
```


### React JS Client (`next-client`)

#### Project setup
```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn dev
```

#### Compiles and minifies for production
```
yarn build
```

### Node Socket.io Server (`server`)

#### Project setup
```
npm install
```

#### Development

```
npm run dev
```

#### Production

```
npm run start
```
