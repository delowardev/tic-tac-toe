# Tic Tac Toe Multiplayer Game
[Live Vue Client: https://tic-tac-toe-vue.now.sh](https://tic-tac-toe-vue.now.sh)

# vue-client

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
